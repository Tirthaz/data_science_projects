# 📊 Data Cleaning & Preprocessing - Insurance Dataset

## 📌 Overview
This project focuses on data preprocessing and cleaning of an insurance dataset.  
The goal is to prepare the data for further analysis and machine learning by handling missing values, encoding categorical variables, and removing outliers.

---

## 🚀 Key Operations Performed

### 🔹 Handling Missing Values
- Identified null values in the dataset
- Removed or treated missing data to ensure consistency

### 🔹 Label Encoding
- Converted categorical columns (e.g., gender, smoker, region) into numerical format
- Used label encoding for better compatibility with machine learning models

### 🔹 Outlier Detection & Removal
- Detected outliers using visualization techniques (boxplots)
- Removed extreme values to improve data quality
